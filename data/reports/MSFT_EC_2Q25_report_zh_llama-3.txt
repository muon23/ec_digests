**執行摘要**
微軟（Microsoft）公布了強勁的財務和營運表現，收入超過 690 億美元，且 Microsoft Cloud 的收入達到 409 億美元，同比增長 21%。公司的人工智慧（AI）業務收入達到年度運營率 130 億美元，同比增長 175%。Microsoft Cloud 的毛利率百分比同比下降 2 個百分點至 70%，而公司的整體毛利率百分比則同比略微增加至 69%。營運費用同比增長 5%，低於預期，而營運利潤率則同比增加 2 個百分點至 45%。

**營運亮點**

* Azure资料中心容量在過去三年內增加了一倍以上，去年錄得創紀錄的新增容量。
* 微軟旗下的數據分析產品Microsoft Fabric已有超過 19,000 名付費客戶，成為公司歷史上增長最快的分析產品。
* Power BI擁有超過 3,000 萬名月活躍用戶，同比增長 40%。
* Azure OpenAI 應用程式同比增長超過兩倍，推動了 SQL Hyperscale 和 Cosmos DB 的增長。
* GitHub擁有 1.5 億名開發者，過去兩年增長 50%，而 GitHub Copilot 的采用率顯著增加，首週就有超過 100萬名用戶註冊。
* Microsoft 365 Copilot 的客戶采用率加速增長，客戶在過去 18 個月內將座位數增加了超過 10 倍，而日使用強度則同比增長超過 60%。

**事業板塊結果**

* 生產力和商業流程：收入增長 14% 至 294 億美元，主要由 Microsoft 365 商業版推動。
* 智慧雲端：收入增長 19% 至 255 億美元，主要由 Azure AI 服務推動（同比增長 157%）。
* 個人電腦：收入基本保持不變，為 147 億美元，主要由 Windows OEM 預裝和搜尋廣告推動。

**財務亮點**

* 收入：696 億美元，同比增長 12%。
* 毛利金額：同比增長 13%（按恆定貨幣匯率計算為 12%）。
* 營運收入：同比增長 17%（按恆定貨幣匯率計算為 16%）。
* 每股盈餘：3.23 美元，同比增長 10%。
* 商業訂單：同比增長 67%（按恆定貨幣匯率計算為 75%），主要由 OpenAI 的 Azure 承諾推動。
* 商業待履行業績義務：2,980 億美元，同比增長 34%（按恆定貨幣匯率計算為 36%）。

**未來展望**

* 收入增長預計為：
　+ 生產力和商業流程：按恆定貨幣匯率計算為 11-12%。
　+ 智慧雲端：按恆定貨幣匯率計算為 19-20%。
　+ 個人電腦：124-128 億美元。
* Microsoft Cloud 毛利率百分比預計約為 69%，同比下降，因為 AI 基礎設施擴張。
* 商業訂單增長預計為同比基本持平。
* 營運費用預計增長 5-6%（按恆定貨幣匯率計算）。
* 稅率：約 18% 的有效稅率（適用於第三季度）。
* 資本支出預計在第三季度和第四季度保持類似的水平，並在 FY26 年實施較低的增長率，同時將重點轉向短期資產。

**指引和展望**

* 公司預計全財年的收入和營運收入將實現雙位數增長，營運利潤率預計同比略微增加。
* 微軟的人工智慧增長率超出了預期，主要由 Azure AI 和 Microsoft Co-Pilot 推動。
* 公司對於擴張 AI 工作負載感到樂觀，DeepSeq 等創新技術有助於改善相關工作。
* 微軟致力於在去市場策略中實現平衡，優先考慮新的設計優勢和企業中的新興 AI 工作負載。
